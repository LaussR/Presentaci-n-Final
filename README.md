# 🚀 Amazon Aurora — Caso Práctico 3

> Módulo 8: Bases de Datos · AWS Training and Certification

## 📋 Descripción

Presentación web interactiva sobre el diseño e implementación conceptual de una solución de base de datos SaaS usando **Amazon Aurora**, orientada a una empresa de procesamiento de pagos en línea de alto volumen (+1M transacciones/día).

La presentación cubre arquitectura Multi-AZ, escalado automático con Aurora Serverless v2, seguridad en capas (IAM + KMS + VPC) y modelo de costos optimizado.

- Aurora Serverless v2 y escalado de ACUs

- Escalado vertical y horizontal
- Alta disponibilidad y failover automático
- Seguridad: IAM + KMS + VPC
- Modelo de costos SaaS
- Simulación de pico de demanda 30x
